# Text Animation

Character-by-character blur-in text animation. Each character animates from blurred to sharp with staggered timing.

- **Source:** [Sera UI](https://seraui.com)
- **Category:** Text Animations
- **Peer Dependencies:** `motion`

## Usage

```tsx
import { BlurInText } from "@/components/text-animations/text-animation";

export default function Hero() {
  return (
    <BlurInText
      text="Sharp Focus Ahead"
      delay={0.2}
      duration={0.5}
    />
  );
}
```
