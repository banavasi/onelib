# Ticker

Animated number counter with easeOutExpo easing. Counts from zero to a target value with configurable duration and formatting.

- **Source:** [Sera UI](https://seraui.com)
- **Category:** Text Animations
- **Dependencies:** None (pure React)

## Usage

```tsx
import { NumberTicker } from "@/components/text-animations/ticker";

export default function Stats() {
  return (
    <div className="text-5xl font-bold">
      <NumberTicker value={1000} prefix="$" duration={2000} />
    </div>
  );
}
```
