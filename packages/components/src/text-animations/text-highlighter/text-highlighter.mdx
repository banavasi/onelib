# Text Highlighter

Scroll-triggered text highlight animation using IntersectionObserver. Animates a gradient highlight behind text when scrolled into view.

- **Source:** [Sera UI](https://seraui.com)
- **Category:** Text Animations
- **Dependencies:** None (uses IntersectionObserver, pure CSS transitions)
