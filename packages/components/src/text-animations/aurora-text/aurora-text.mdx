# Aurora Text

Gradient animated text with flowing aurora color effect. Uses CSS keyframes for smooth color transitions.

- **Source:** [Sera UI](https://seraui.com)
- **Category:** Text Animations
- **Dependencies:** None (pure CSS animation)

## Usage

```tsx
import { AuroraText } from "@/components/text-animations/aurora-text";

export default function Hero() {
  return (
    <h1 className="text-6xl font-bold">
      <AuroraText
        colors={["#FF0080", "#7928CA", "#0070F3", "#38bdf8"]}
        speed={5}
      >
        Beautiful
      </AuroraText>
    </h1>
  );
}
```
