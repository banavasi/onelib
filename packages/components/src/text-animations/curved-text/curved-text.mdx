# Curved Text

Canvas-based curved scrolling text with configurable direction, speed, and curve. Renders text along a sine-like curve with optional mouse interaction.

- **Source:** [Sera UI](https://seraui.com)
- **Category:** Text Animations
- **Dependencies:** None (pure canvas)

## Usage

```tsx
import { CanvasCurvedLoop } from "@/components/text-animations/curved-text";

export default function MarqueeSection() {
  return (
    <div className="w-full text-white">
      <CanvasCurvedLoop
        text="Scrolling curved text"
        speed={1}
        curveHeight={40}
        direction="left"
        fontSize={48}
        height={200}
      />
    </div>
  );
}
```
