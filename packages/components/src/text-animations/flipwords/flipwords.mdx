# Flip Words

Animated word rotator that cycles through a list of words with blur and slide transitions.

- **Source:** [Sera UI](https://seraui.com)
- **Category:** Text Animations
- **Peer Dependencies:** `motion`

### Fast Rotation

## Usage

```tsx
import { FlipWords } from "@/components/text-animations/flipwords";

export default function Hero() {
  return (
    <span className="text-4xl font-bold">
      We are{" "}
      <FlipWords
        words={["creating", "building", "designing", "crafting"]}
        duration={3000}
      />
    </span>
  );
}
```
