# Liquid Ether

Interactive fluid dynamics simulation with customizable colors and mouse-driven forces. Features BFECC advection, viscous diffusion, and Poisson pressure solver. Uses Three.js WebGL.

- **Source:** [ReactBits](https://reactbits.dev)

## Usage

Copy the component into your project at `src/components/backgrounds/liquid-ether/liquid-ether.tsx`:

```tsx
// Install peer dependency: pnpm add three
import LiquidEther from "@/components/backgrounds/liquid-ether";

export default function Example() {
  return (
    <div className="relative w-full h-screen">
      <LiquidEther
        colors={["#FF6B35", "#FFD700", "#FF1493"]}
        mouseForce={20}
        autoDemo={true}
      />
    </div>
  );
}
```
