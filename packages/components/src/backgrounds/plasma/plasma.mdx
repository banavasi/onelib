# Plasma

Animated plasma effect with customizable color tinting, direction control, and mouse interaction. Uses OGL WebGL2 fragment shaders for real-time plasma generation.

## Usage

Copy the component into your project at `src/components/backgrounds/plasma/plasma.tsx`:

```tsx
// Install peer dependency: pnpm add ogl
import Plasma from "@/components/backgrounds/plasma";

export default function Example() {
  return (
    <div className="relative w-full h-screen">
      <Plasma
        color="#8B5CF6"
        speed={1.5}
        direction="forward"
      />
    </div>
  );
}
```
