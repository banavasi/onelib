# Particles

3D particle system with customizable colors, density, and optional hover interaction. Uses OGL WebGL renderer with point-based geometry and simplex noise animation.

- **Source:** [ReactBits](https://reactbits.dev)

## Usage

Copy the component into your project at `src/components/backgrounds/particles/particles.tsx`:

```tsx
// Install peer dependency: pnpm add ogl
import Particles from "@/components/backgrounds/particles";

export default function Example() {
  return (
    <div className="relative w-full h-screen">
      <Particles
        particleCount={200}
        speed={0.1}
        particleColors={["#FF6B35", "#00BFFF", "#7cff67"]}
      />
    </div>
  );
}
```
