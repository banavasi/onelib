# Pixel Snow

Three.js pixel-based snow shader with configurable flake shape, density, and wind direction. Renders volumetric snowfall using raymarching through a 3D voxel grid with square, round, or detailed snowflake shapes.

- **Source:** [ReactBits](https://reactbits.dev)
- **Category:** Backgrounds

## Usage

Copy the component into your project at `src/components/backgrounds/pixel-snow/pixel-snow.tsx`:

```tsx
// Install peer dependency: pnpm add three
import PixelSnow from "@/components/backgrounds/pixel-snow";

export default function Example() {
  return (
    <div className="relative w-full h-screen bg-black">
      <PixelSnow
        variant="snowflake"
        density={0.3}
        speed={1.25}
      />
    </div>
  );
}
```
