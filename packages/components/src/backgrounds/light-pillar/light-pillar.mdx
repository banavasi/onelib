# Light Pillar

Volumetric light pillar effect with customizable gradient colors, glow amount, rotation speed, and quality settings. Uses Three.js WebGL shaders with ray marching.

## Usage

Copy the component into your project at `src/components/backgrounds/light-pillar/light-pillar.tsx`:

```tsx
// Install peer dependency: pnpm add three
import LightPillar from "@/components/backgrounds/light-pillar";

export default function Example() {
  return (
    <div className="relative w-full h-screen">
      <LightPillar
        topColor="#00FF88"
        bottomColor="#0044FF"
        intensity={1.2}
        rotationSpeed={0.3}
      />
    </div>
  );
}
```
