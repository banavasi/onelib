# Prism

Animated 3D prism/pyramid effect rendered via ray marching with customizable rotation, hover interaction, glow, and color shifting. Uses OGL WebGL shaders.

- **Source:** [ReactBits](https://reactbits.dev)
- **Category:** Backgrounds

## Usage

Copy the component into your project at `src/components/backgrounds/prism/prism.tsx`:

```tsx
// Install peer dependency: pnpm add ogl
import Prism from "@/components/backgrounds/prism";

export default function Example() {
  return (
    <div className="relative w-full h-screen">
      <Prism
        animationType="hover"
        glow={1.2}
        hueShift={0.5}
      />
    </div>
  );
}
```
