# Grid Scan

Three.js + postprocessing background with 3D grid scanner effect. Features animated scan lines, bloom, chromatic aberration, and optional webcam-based face tracking for parallax.

- **Source:** [ReactBits](https://reactbits.dev)
- **Category:** Backgrounds
- **Peer Dependencies:** `three`, `postprocessing`, `face-api.js` â€” install with `pnpm add three postprocessing face-api.js`

## Usage

Copy the component into your project at `src/components/backgrounds/grid-scan/grid-scan.tsx`:

```tsx
// Install peer dependencies: pnpm add three postprocessing face-api.js
import { GridScan } from "@/components/backgrounds/grid-scan";

export default function Example() {
  return (
    <div className="relative w-full h-screen">
      <GridScan
        scanColor="#FF9FFC"
        linesColor="#392e4e"
        scanDirection="pingpong"
      />
    </div>
  );
}
```
