# Dark Veil

Neural network-generated dark abstract background with optional hue shifting, scanlines, noise grain, and warp distortion effects. Uses OGL WebGL shaders with a CPPN (Compositional Pattern Producing Network).

- **Source:** [ReactBits](https://reactbits.dev)
- **Category:** Backgrounds

## Usage

Copy the component into your project at `src/components/backgrounds/dark-veil/dark-veil.tsx`:

```tsx
// Install peer dependency: pnpm add ogl
import DarkVeil from "@/components/backgrounds/dark-veil";

export default function Example() {
  return (
    <div className="relative w-full h-screen">
      <DarkVeil
        hueShift={45}
        speed={0.5}
        scanlineIntensity={0.3}
      />
    </div>
  );
}
```
