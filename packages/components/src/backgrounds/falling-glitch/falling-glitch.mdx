# Falling Glitch

Canvas-based falling glitch text effect with configurable colors, speed, and intensity. Characters fall and randomly glitch across the canvas with a vignette overlay.

- **Source:** [Sera UI](https://seraui.com)
- **Category:** Backgrounds
- **Peer Dependencies:** None

## Usage

```tsx
import { FallingGlitch } from "@/components/backgrounds/falling-glitch";

export default function Example() {
  return (
    <div className="relative w-full h-screen">
      <FallingGlitch
        glitchColors={["#ff7cce", "#7cf0ff", "#fcf07c"]}
        fallSpeed={0.75}
        outerVignette
      >
        <h1 className="text-4xl font-bold text-white">Welcome</h1>
      </FallingGlitch>
    </div>
  );
}
```
