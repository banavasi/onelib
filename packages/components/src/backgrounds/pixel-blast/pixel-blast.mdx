# Pixel Blast

Animated pixel/dithered pattern background with click ripples, multiple shape variants, and optional liquid distortion. Uses Three.js with postprocessing.

- **Source:** [ReactBits](https://reactbits.dev)

## Usage

Copy the component into your project at `src/components/backgrounds/pixel-blast/pixel-blast.tsx`:

```tsx
// Install peer dependencies: pnpm add three postprocessing
import PixelBlast from "@/components/backgrounds/pixel-blast";

export default function HeroSection() {
  return (
    <div className="relative w-full h-screen">
      <PixelBlast
        variant="square"
        pixelSize={3}
        color="#B19EEF"
        speed={0.5}
        enableRipples={true}
      />
    </div>
  );
}
```
