# Video Text

Renders text with a video playing inside the letterforms using SVG clip-path masking. Configurable typography, video source, and text positioning.

- **Source:** [Sera UI](https://seraui.com)
- **Category:** Sections
- **Peer Dependencies:** None

## Usage

```tsx
import { VideoText } from "@/components/sections/video-text";

export default function HeroSection() {
  return (
    <div className="w-full h-screen bg-black">
      <VideoText src="/hero-video.mp4" fontSize="clamp(4rem, 15vw, 12rem)">
        HELLO
      </VideoText>
    </div>
  );
}
```
