# Orbiting Skills

Orbital animation visualization showing tech icons orbiting around a center element. Features inner and outer orbit rings with counter-rotating animations. Self-contained with inline SVG icons and CSS keyframe animations.

- **Source:** [Sera UI](https://seraui.com)
- **Category:** Sections
- **Peer Dependencies:** None
