# Basic Accordion

A minimalistic accordion component with plus icon that rotates 45deg on open. Manages single-open state with memo'd items for performance.

- **Source:** [Sera UI](https://seraui.com)
- **Category:** Accordions
- **Dependencies:** None (React + Tailwind only)

## Usage

Copy the component into your project at `src/components/accordions/basic-accordion/basic-accordion.tsx`:

```tsx
import { BasicAccordion } from "@/components/accordions/basic-accordion";

const faqItems = [
  { title: "Question one?", content: "Answer one." },
  { title: "Question two?", content: "Answer two." },
];

export default function FAQ() {
  return <BasicAccordion items={faqItems} />;
}
```
