# Liquid Glass Button

Apple-style liquid glass button with SVG displacement map distortion filter. Supports subtle, default, bold, and ghost presets with size variants.

- **Source:** [Sera UI](https://seraui.com)
- **Category:** Buttons
- **Peer Dependencies:** `class-variance-authority`
- **Dependencies:** `clsx`, `tailwind-merge` (included in template)

## Usage

Install peer dependencies:

```bash
npm install class-variance-authority
```

Copy the component into your project at `src/components/buttons/liquid-glass-button/liquid-glass-button.tsx`:

```tsx
import { LiquidGlassButton } from "@/components/buttons/liquid-glass-button";

export default function HeroSection() {
  return (
    <div className="relative bg-cover bg-center h-screen" style={{ backgroundImage: "url('/hero.jpg')" }}>
      <LiquidGlassButton variant="bold" size="lg">
        Get Started
      </LiquidGlassButton>
      <LiquidGlassButton variant="ghost" size="sm">
        Learn More
      </LiquidGlassButton>
    </div>
  );
}
```
