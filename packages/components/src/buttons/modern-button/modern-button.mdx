# Modern Button

Two modern button components: **CopyButton** with animated jiggle on copy, and **ProcessingButton** with idle/processing/success/error state transitions.

- **Source:** [Sera UI](https://seraui.com)
- **Category:** Buttons
- **Dependencies:** `clsx`, `tailwind-merge` (included in template)

## Usage

Copy the component into your project at `src/components/buttons/modern-button/modern-button.tsx`:

```tsx
import { CopyButton, ProcessingButton } from "@/components/buttons/modern-button";

export default function MyPage() {
  const handleProcess = async () => {
    await fetch("/api/process");
  };

  return (
    <div className="flex gap-4">
      <CopyButton textToCopy="npm install @oneorigin/onelib" />
      <ProcessingButton onProcess={handleProcess}>
        Download
      </ProcessingButton>
    </div>
  );
}
```
