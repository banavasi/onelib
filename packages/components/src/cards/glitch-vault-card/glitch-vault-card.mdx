# Glitch Vault Card

A card component with matrix/glitch text effect on hover. Features a particle system with object pooling, offscreen canvas rendering, and configurable performance modes for smooth animation.

- **Source:** [Sera UI](https://seraui.com)
- **Category:** Cards
- **Dependencies:** None (React + Tailwind only)

## Usage

Copy the component into your project at `src/components/cards/glitch-vault-card/glitch-vault-card.tsx`:

```tsx
import { GlitchVaultCard } from "@/components/cards/glitch-vault-card";

export default function Example() {
  return (
    <GlitchVaultCard
      className="max-w-sm bg-black/20 border border-white/30 rounded-2xl"
      glitchColor="#0AF0F0"
      glitchRadius={120}
    >
      <div className="p-8 text-center">
        <h2 className="text-2xl font-bold text-white">Title</h2>
        <p className="text-gray-300">Hover to see the effect</p>
      </div>
    </GlitchVaultCard>
  );
}
```
